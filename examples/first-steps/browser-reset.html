Basic reset

This ensures the page displays the same in all browsers.

Add CSS to the Universal selector *

###

/* universal selector, selects all elements on page */

* {
  /* removes margins and padding from all elements 
  like headers and paragraphs*/
  margin: 0;
  padding: 0;
  /* changes box model so borders and padding are not added to 
  total width or heigh specified for that box*/
  box-sizing: border-box;
}

/* set fonts on body so it's inherited in rest of page */
body {
  font-family: "Lato", sans-serif;
  font-weight: 400;
  font-size: 16px;
  line-height: 1.7;
  color: #777;
}
