Text gradient

Creates a gradient in the text color.

Step 1 - Add a gradient background color

.heading-secondary {
  font-size: 3.5rem;
  text-transform: uppercase;
  font-weight: 700;

  /* Set gradient on text */
  /* first set to inline-block so it doesn't take up 100% of width*/
  display: inline-block;

  /* first set background color of block */
  background-image: linear-gradient(to right, #7ed56f, #162adb);

  /* background clipped to where text is */
  -webkit-background-clip: text;

  /* show the text, not the background gradient 
  You cannot see the gradient yet because the text now sits on top of background
  make text transparent so you can see background color */
  color: transparent;

  letter-spacing: 2px;
  transition: all .2s;
}

<h2 class="heading-secondary">
  Existing tours for adventurous people
</h2>
