Center a section

To horizontally center a block element (like <div>), use margin: auto;

Make sure to also add a width. Center aligning has no effect if the width property is not set (or set to 100%).

Adding a width will cause the remaining space to be split equally in both margins.

.center {
  margin: auto;
  width: 60%;
}

Center text in a section

Create a utility class (Sass) that can be reused.

This works if the h2 is an inline-block element.

If that parent element is set to text-align: center, the text in the inline-block element will be centered.

.heading-secondary {
  display: inline-block;
}
            
.u-center-text {
text-align: center;
}


<div class="u-center-text u-margin-bottom-big">
  <h2 class="heading-secondary">
      Existing tours for adventurous people
  </h2>
</div>
