Absolute

The parent element must be set to relative.

.blog-post {
  width: 75%;
  position: relative;
  border: 1px solid cyan;
}

.date {
  position: absolute;
  top: 0; /*right at top with zero pixels between element*/
  right: 0;/*no space between container and element*/
}

<div class="parent-div">
  <p>Blog post section</p>
  <p>Blog post section</p>
  <p>Blog post section</p>
  <p class="date">This appears on the top right.</p>
</div>
