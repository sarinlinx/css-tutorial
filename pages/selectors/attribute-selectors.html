Attribute selectors

The attribute selector locates elements with a specified attribute. Here are a few examples.

Example using the 'target' attribute

The target attribute 
Make sure to <a href="#" target="_blank">Click Here</a>

Target the attribute with CSS.

[target] {
  color: yellow;
}

This would make the entire link text yellow.

Attribute and value selector

This selects the 'target' attribute only if the value is '_blank'. 
Make sure you <a href="#" target="_blank">Click Here</a>

Target the attribute with CSS.

[target=_blank] {
  color: green;
}

This would make only the link with a target of _blank green.
