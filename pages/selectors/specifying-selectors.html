Specifying element selectors

You can specifically assign a class to only one HTML element using a period to separate them.

This CSS applies only to h1 elements with a class of 'center'.

h1.center {
      text-align: center;
    }

Descendent selectors

Use multiple selectors, separated by a space to match the descendent elements.

Example: Select a link in a paragraph and change the color to red.

<p>Please click <a href="#">HERE!!!</a></p>

        p a {
          color: red;
        }

        
Please click HERE!!!

Multiple descendent selectors

Select paragraphs and links inside <section>

<section>
          <p>Please click <a href="#">HERE!!!</a></p>
        </section>

            section p a {
          color: blue;
        }
            

            
Please click HERE!!!

Descendant selectors

Descendant selectors add a space between the selectors. This makes it possible to match all elements that are descendants of a specified element.

This example looks inside all <div> elements and selects any <p> elements inside. 
div p {
      background-color: yellow;
    }
You can also skip levels. This example looks inside all <ancestor> elements and selects any <descendant> elements inside. 
<div class="ancestor">
      <div class="parent">
        <div class="child">This is not styled</div>
          <div class="descendant">This IS styled</div>
        </div>
      </div>
    </div>
CSS to select the descendant. 
    .ancestor .descendant {
      color: blue;
    }
Result: 
This is not styled
    This IS styled

Here is another example that looks inside any <section> elements for <a> elements.

<section>
      <div class="example">
        <p>Link goes here <a href="#">link</a>.</p>
      </div>
    </section>
CSS to select the anchor tag within the section tag. 
    section a {
      /*styles*/
    }

Child selectors

Select ONLY the child element using > symbol between parent and child elements.

This example ONLY looks in the <section> element for a direct <a> element. 

<section>
      <a href="#">child 1</a>
      <a href="#">child 2</a>

      <div>
        <a href="#">descendant</a>
      </div>
    </section>

CSS to select the first two anchor tag children inside the section tag. The 3rd isn't a child so it's not select.

    section > a {
      color: green;
    }
Result: 
child 1
    child 2
    descendant

Adjacent Sibling Selectors

This example ONLY selects the paragraph element directly after an <h2> element.

Adjacent Sibling Selectors only select the element immediately following a specified element. They are separated with a + symbol.

<p>Paragraph 1</p>

    <h2>Heading</h2>
    <p>Paragraph 2</p>
    <p>Paragraph 3</p>
CSS to select the paragraph immediately following the h2 tag. The 3rd doesn't immediately follow so it's not selected. 
    h2 + p {
      color: blue;
    }
Result: 
Paragraph 1
    Heading
    Paragraph 2
    Paragraph 3

General Sibling Selector

General Sibling Selectors select all specified elements following a specified element. They are separated with a ~ symbol.

This example selects all paragraphs following the <div> element.

<p>Paragraph 1.</p>
    <div>
      <p>Paragraph 2.</p>
    </div>
    <p>Paragraph 3.</p>
    <p>Paragraph 4.</p>
CSS to select only the paragraphs that follow the div tag. 
div ~ p {
      color: red;
    }

Result:

Paragraph 1
    Paragraph 2
    Paragraph 3
    Paragraph 4
