Pseudo-Class Selectors

pseudo selectors are keywords added after a selector with a colon symbol.

Example

Change the color when hovering over a link.

a:hover {
  color: green;
}

Hover over this link to see it change to green.

Pseudo Element Selectors

pseudo element selectors select certain parts of element that are not part of the HTML.

pseudo element selectors use double colon notation to differentiate between peseudo class selectors.

NOTE: Older versions of CSS use a single colon. Modern browser will work with either.

::before and ::after

:before and :after used in box-model-fix. They are also used to GENERATE elements inserted before or after the selected element

These generated elements do NOT become part of the HTML DOM.

These must always be used with the content property.

.someClass::before {
  content: "words for the pseudo selector ";
}

This is the normal text in the HTML.

Using ::before and ::after to add quote marks

If you need to put quotes around quoted text, you just need to escape the double quote.

You could also put the double quote inside of single quotes , or use unicode '\201C'

.someClass::before {
  color: red;
  content: "\"";
}

.someClass::after {
  color: red;
  content: "\"";
}

Red quotes are added.

Selecting first and last children of parent elements

Apply a pseudo selector to a child element of a parent to specifically target that child.

This must be added to the child element.

Select the first and last child elements from the parent.

NOTE how the first-child and last-child selectors are added to the child elements, not the parent.

<section>
      <p>Paragraph</p> <!-- this is red -->
      <p>Paragraph</p>
      <p>Paragraph</p> <!-- this is blue -->
    </section>

    p:first-child {
      color: red;
    }

    p:last-child {
      color: blue;
    }

        

    Paragraph <!-- this is red -->
    Paragraph
    Paragraph <!-- this is blue -->

Selecting first and last type from a parent element.

This is different from first-child last-child since it doesn't matter what order the HTML is in.

The first type of the specified element will be selected.

Select the first and last paragraph elements in a parent.

Notice how the paragraph tags are not first or last inthe HTML, but this pseudo element allows you to target them since they are the first and last of that specific type of element.

<section>
      <p>Paragraph</p> <!-- this is red -->
      <p>Paragraph</p>
      <p>Paragraph</p> <!-- this is blue -->
      <a href="#">link</a>
    </section>

    p:first-of-type {
      color: red;
    }

    p:last-of-type {
      color: blue;
    }
