Setting a background image

Add a background image.

body {
  background-image: url("paper.gif");
}

To specify multiple images, separate the URLs with a comma.

body {
  background-image: url("image1.png"),
  url("image2.png");
}

Background repeat

By default, the background-image property repeats an image both horizontally and vertically.

The image is repeated horizontally only. If the image has a short height, anything below is inherits its background color.

Do not repeat the image

This forces the image to not repeat at all.

      background: url("../images/backgrounds/examples/purple-dot.jpg");
      background-repeat: no-repeat;
    }

The result is that the image shows once in the top left corner.

Background attachement

By default, the background image will not move with the page.

The background attachment property allows you to control what happens to the background image as the page is navigated.

Fixed — Fix the image to the page as you scroll

This causes the image to be fixed to the top left of the page. As the user scrolls, the image always remains on the top left and never goes out of sight.

body {
      background: url("../images/backgrounds/examples/purple-dot.jpg");
      background-repeat: no-repeat;
      background-attachment: fixed;
    }

This only works on the body element. If you used 'fixed' for any other element, the image will disappear. This is because 'fixed' is relative to the viewport, not the element you're adding the code to.

Allow the image to scroll with the page

This causes the image to scroll with the page. As the user scrolls, the image scrolls also. This is the default behavior.

body {
      background: url("../images/backgrounds/examples/purple-dot.jpg");
      background-repeat: no-repeat;
      background-attachment: scroll;
    }

This causes the image to scroll with the page. As the user scrolls, the image scrolls also. This is the default behavior.

Background Position

Position the background image to a specific area of the page.

Using positional words

You can use words like 'top right' to position the background.

body {
      background: url("../images/backgrounds/examples/purple-dot.jpg");
      background-repeat: no-repeat;
      background-position: top right;
    }

The result will be the image on the top right.

Other options are:

  • left top
  • left center
  • left bottom
  • right top
  • right center
  • right bottom
  • center top
  • center center
  • center bottom

If you only specify one keyword, the other value will be "center"

Using % values

You can use % values to position the background.

The first value is the horizontal position and the second value is the vertical. The top left corner is 0% 0%. The right bottom corner is 100% 100%.

If you only specify one value, the other value will be 50%. Default value is: 0% 0%.

body {
      background: url("../images/backgrounds/examples/purple-dot.jpg");
      background-repeat: no-repeat;
      background-position: 20% 10%;
    }

This example moves the image 20% to the right and 10% down on the page.

Using CSS units

You can position using any CSS unit like pixels.

body {
      background: url("../images/backgrounds/examples/purple-dot.jpg");
      background-repeat: no-repeat;
      background-position: 20px 50px;
    }

This example moves the image 20 pixels to the right and 50 pixels down on the page.

Different background images for different displays

If you have a high resolution display (retina) you may want to display a different image than a smaller screen with less resolution.

Use a media query.

header {
          background: url(image.jpg);
        }

        @media (min-resolution: 192dpi) {
          header {
            /* Using a larger image for retina screens */
            background: url(image@2x.jpg);
          }
        }
