Opacity

The opacity property specifies the opacity/transparency of an element. It can take a value from 0.0 - 1.0. The lower value, the more transparent:

When using the opacity property to add transparency to the background of an element, all of its child elements inherit the same transparency. This can make the text inside a fully transparent element hard to read.

.image-bg {
background-color: green;
opacity: 0.3;
}

Background transparency

If there is already a background added, you can add a second layer over it. You can then adjust this second layer's transparency value.

Make sure this is NO SPACE between rgba and the parenthese.

<div class="image-bg">
  <p class="rgba-transparency">Testing to see how the transparency works.</p>
</div>

.image-bg {
  background: url("images/note.jpg")
}

.rgba-transparency {
  background: rgba(199, 21, 133, .25);
}

The result will be a transparent layer over top the current background.

Linear gradient background

This shows how to add a background of 2 colors that blend into each other.

<p class="linear-gradient">Testing to see how the linear gradient works.</p>

    .linear-gradient {
      background: linear-gradient(red, green)
    }

The result will be a transparent layer over top the current background.

Testing to see how the linear gradient works.

You can also add a 3rd color.

<p class="linear-gradient-3colors">Testing to see how the linear gradient works.</p>
        

    .linear-gradient-3colors {
      background: linear-gradient-3colors(red, yellow, green),
    }

The result will be a transparent layer over top the current background.

Testing to see how the linear gradient works.

Transparent linear gradient over a background image

This shows how to add a transparent linear gradient of 3 colors over a background image.

The linear-gradient values must come first.

<div class="image-bg">
      <p class="transparent-linear-gradient">Transparent linear gradient over an background image.</p>
    <div>
    

    .transparent-linear-gradient {
      height: 40px;
      background: linear-gradient(rgba(255, 0, 0, 0.35),  /*red*/
      rgba(255, 255, 0, 0.35), /*yellow*/
      rgba(0, 0, 255, 0.35)),  /*blue*/
      url("images/backgrounds/examples/white-bg.jpg");
    }

The result will be a transparent gradient over a background image.
