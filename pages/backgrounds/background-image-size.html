Background size

Background size specifies the size of background images.

There are four different syntaxes you can use with this property:

  • Keyword syntax ("auto", "cover" and "contain")
  • One-value syntax (sets the width of the image (height becomes "auto")
  • Two-value syntax (first value: width of the image, second value: height)
  • Multiple background syntax (separated with comma).

Keyword syntax

auto is the default value. The background image is displayed in its original size.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat:no-repeat;
  background-size:auto;
}

One value syntax

If you set only one value for width, the second value for height becomes 'auto'.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat:no-repeat;
  background-size: 80px;
}

Example: The image is set to 80px.

You can also use percentage, but this is not recommended. This isn't a good idea because the aspect ratio does not change, so there could be blank space at the bottom of the page after the image.

Two value syntax

This sets both a width and height to the property.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat:no-repeat;
  background-size: 80px 120px;
}

Example: The image is set to 80px wide and 120px high.

You can also use percentage.

Shorthand

You can use shorthand to list all background CSS properites in a single declaration.

When using the shorthand property, the recommended order of the property values is:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

But the order doesn't matter.

If you mix shortland with full declarations, shorthand will override.

body {
  background: #ffffff url("img_tree.png") no-repeat scroll right top;
}

If you include background-size, it MUST be after background-position and separated with /.

body {
  background: url(image.png) no-repeat fixed 0px 0px / cover;
}

Shorthand for multiple images

Longhand for multiple image styles.

body {
  background-image: url(image1.png), url(image2.png);
  background-repeat: no-repeat, no-repeat;
  background-position: top right, bottom left;
}

Shorthand for multiple image styles.

body {
  background: url(image1.png) no-repeat top right,
              url(image2.png) no-repeat bottom left;
}
