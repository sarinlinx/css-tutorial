Vendor prefixes

Each browser uses its own prefix applied to the beginning of the property name.

Vendor prefixes are used to add browser support for new or experimental features that are not fully standardized across browsers yet.

Android, iOS, Chrome, Safari

-webkit-

Firefox

-moz-

Internet Explorer

-ms-

Opera

-o-

How to use Vendor Prefixes?

CSS is rendered top to bottom. Because of this, you should list the Vendor Prefix versions of your code first.

Only after the Vendor Prefix lines have been written should you add the standard CSS property.

Browsers will read the first line and ignore it if it's not the prefix for that specific browser.
