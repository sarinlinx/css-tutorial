Reduce image size

To adjust the size of an image, add a class. Then, in CSS, add only the width value. The height automatically adjusts to aspect-ratio.

<img class="blah" asdf>

.blah {
  width: 300px;
}

Reshape image

Add border radius. Make sure you start with a square image or it will turn it into an oval.

img {
  border-radius: 50%;
}

Here's an example using pixels.

img {
  border-radius: 22px;
}

Center an image

Use margin:auto to center and image.

img {
      border-radius: 50%;
      }

Here's an example using pixels.

img {
  display: block;
  margin: 0 auto;
}
